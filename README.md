🩺 Detecting Pneumonia from Chest X-Ray Images Using Deep Learning
This project uses Convolutional Neural Networks (CNNs) and Transfer Learning with ResNet architectures to classify chest X-ray images as Pneumonia or Normal. The notebook compares the performance of a custom CNN model with pretrained ResNet50V2 and ResNet101V2 models.
📌 Project Overview
Pneumonia is a serious respiratory infection that can be detected through chest X-ray imaging. This project applies deep learning techniques to automate the detection process and evaluate different neural network architectures for image classification.
The models implemented include:
Custom CNN
ResNet50V2 (Transfer Learning)
ResNet101V2 (Transfer Learning)
